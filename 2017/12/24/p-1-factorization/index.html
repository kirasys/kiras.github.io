p-1 factorization

p-1 factorization 알고리즘은 N = pq (p,q 는 소수) 에서 N을 소인수 분해하는 알고리즘이다.

Concept

p가 소수일 때 ap-1 = 1 (mod p) 를 만족한다. 마찬가지로 ak(p-1) = 1 이므로 ak(p-1) - 1 은 p의 곱으로 표현할 수 있다. k(p-1)을 알 수 없으므로 2!, 3!, 4! … n!(n < B-smooth) 사용하고 gcd(ak(p-1) - 1,N) 을 계산하여 인수를 찾는다. 그래서 p-1 이 smooth number일 때 사용하기 적절하다.

Algorithm

a,B = 2,10000
gcd_result,i = 1,1

while gcd_result<=1:
    i += 1
    if i>B:
        break
    a = pow(a,i,n)
    gcd_result = GCD(a-1,n)
